# LinkedIn Content Strategy

## When to Use
When planning, writing, or reviewing LinkedIn content for Muhammad Nurunnabi. Load this skill before any content production session or cron-triggered content creation.

## Voice Profile — Muhammad Nurunnabi (Personal)

**Tone:** Technical operator, direct, first-person.
**Style:** Build-in-public. Shares what worked AND what failed. No fake vulnerability, no humble-brag.
**Signature moves:** Real numbers, actual product names (Amanda, Layla, Bold Business, DhabiVoice), specific markets (UAE, Malaysia, KSA).
**Format:** Short paragraphs, strategic line breaks, emoji-light (max 2-3 per post, never in the hook).
**Contractions:** Yes — I'm, don't, can't, we've, it's. Write like you talk.
**Sentence length:** Mix aggressively. 5-word punches next to 25-word explanations.

### Words to NEVER use
- leverage → use
- cutting-edge → name the specific tech
- game-changer → state the actual impact with a number
- in today's rapidly evolving landscape → DELETE
- it's worth noting → DELETE
- let's dive in / let's unpack → DELETE
- the landscape of → DELETE
- revolutionary → DELETE
- synergy → DELETE
- at the end of the day → DELETE
- moving forward → DELETE

### Banned phrase patterns
- "I'm excited to announce..." → Just announce it
- "Thrilled to share..." → Just share it
- "After much deliberation..." → Get to the point
- "Here's the thing..." → Just say the thing
- Any sentence starting with "As a [role]..."

## Post Structures (proven high-engagement formats)

### 1. Hook-Story-Lesson
```

[2-3 paragraph story with specific details]

[Clear takeaway — 1-2 sentences]

[Question CTA that invites real discussion]

```

### 2. List Post

### 3. Contrarian Take
```

[Evidence from personal experience — 3-4 paragraphs]

[Nuanced conclusion — not just "the opposite is true" but "the reality is more complex"]

#hashtag1 #hashtag2 #hashtag3
```

### 4. Behind-the-Scenes
```

[What I learned — honest, not performative]

[Practical takeaway others can use]

#hashtag1 #hashtag2 #hashtag3
```

### 5. Micro-Tutorial
```

...
Step 5: [Actionable instruction]

Tool I used: [specific tool name]
Time it took: [specific number]

Save this for later. What would you add?

#hashtag1 #hashtag2 #hashtag3
```

## Algorithm Optimization Rules

1. **Hook is everything.** LinkedIn truncates at ~210 characters. The hook must create curiosity or tension in that window.
2. **Line breaks = readability.** Break every 1-2 sentences. Dense paragraphs die on mobile.
3. **1200-1800 characters** is the sweet spot for feed posts. Over 3000 gets "see more" fatigue.
4. **3-5 hashtags max.** Specific beats generic (#VoiceAI > #AI, #RetellAI > #Tech).
5. **End with a question.** Comments are the #1 algorithm signal in 2025-2026. A good question drives 3-5x more comments than "like if you agree."
6. **Post between 7-9 AM ET** for US professional audience.
7. **Carousels get 2-3x more reach** than text-only posts. Use them for Wednesday content.
8. **Video under 90 seconds**, native upload (never a YouTube link).
9. **No external links in the post body** — LinkedIn suppresses reach for posts with outbound links. Put links in comments.
10. **Golden hour:** The first 60 minutes determine distribution. Engage with every comment immediately.
